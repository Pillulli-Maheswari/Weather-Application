## Description
A responsive web application that displays real-time weather information for any city. Users can view current weather, forecasts, and air quality using WeatherAPI.

## Technologies Used

- CSS
- JavaScript
- WeatherAPI

## Features
- Search for weather in any city
- Display current weather, 3-day forecast, and air quality
- Responsive design for desktop and mobile
- Dynamic and interactive UI
